[Install Driver]
Since the operating system after Win7 SP1 already supports the SHA-2 secure hash algorithm for signing, SHA-1 and SHA-2 Drivers are provided;
When installing, users only need to select the upper layer \MLink_Virtual_COM_Port_Driver to install.

The operating system will automatically select the appropriate Driver to install

[安裝 Driver]
由於Win7 SP1 後的作業系統已支援SHA-2 安全雜湊演算法用於簽章，故提供SHA-1和SHA-2的Driver；
用戶在安裝時只需選擇上一層的\MLink_Virtual_COM_Port_Driver 去安裝既可，
作業系統會自動選擇適合的Driver去安裝
